Supports per page themes, add-on subthemes, and theme updating for the Aspen Theme.

== Description ==

This plugin provides several enhancements for the Aspen WordPress theme.

= Add-on Themes =

This plugin also allows you to upload and install new, add-on subthemes. There are free and premium add-on subthemes available
at http://aspentheme.com.

= Theme Updating =

Normally, you will be able to update your version of the Aspen theme directly from WordPress.org.
But sometimes there is a considerable delay between the time a new version of the theme is submitted
to WordPress.org, and when it becomes available there. This option lets you easily update Aspen after
downloading it from the Aspen website.

= Per Page Theme Switching =

You can save your subthemes in the WordPress database. This allows Per Page Theme setting by specifying
'aspen_per_page_theme' as a Custom Field.

= Clear Aspen Settings =

You can clear Aspen theme settings from the WP database.

== Installation ==

It is easiest to use the standard WordPress.org Plugin Add Plugin page.
